🧠 Reasoning & Logic

Include Chain of Thought / Thinking

Instructs the AI to 'think step-by-step' before answering, which improves accuracy on complex reasoning tasks.

Use Hidden Self-Review Rubric

Inserts a rule for the AI to create an internal quality rubric and iterate on its answer until it meets a high standard. Very effective for creative or code-gen tasks.

Encourage Agentic Proactivity

Adds a persistence instruction, telling the agent to continue working until a task is fully resolved without asking for permission to proceed.

📚 Grounding & Factuality

Allow "I don't know" answers

Gives the AI explicit permission to state it lacks information, which is a key technique for reducing factual hallucinations.

Restrict to provided documents

Instructs the AI to base responses only on provided documents, preventing outside knowledge contamination in RAG scenarios.

Require citations / claim verification

Demands proper attribution and source references for all factual claims, improving accountability and verifiability.

Extract supporting quotes first

Uses the "quotes-first" technique: AI finds relevant quotes from documents before generating responses, dramatically improving factual accuracy.

Optimize for Long Context

Places large documents first in the prompt for better recall with long-context models. Ideal for processing extensive reference materials.

💻 Code Generation

Include general code editing rules

Adds best practices for code quality, readability, error handling, and modularity. Essential for programming tasks.

Encourage parallel tool calling

Instructs the AI to use multiple tools simultaneously when possible, improving efficiency for complex workflows.

Ask to clean up temp files

Adds file management instructions to maintain organized project structure and remove temporary artifacts.

📝 Formatting & Structure

Use XML tags for structure

Automatically wraps inputs like context and examples in XML tags (e.g., <context>) to improve the AI's ability to parse the prompt.

Prefer JSON output

Instructs the AI to structure responses as valid JSON when appropriate, useful for data processing and API integration.

Forbid Markdown in output

Prevents Markdown formatting in responses, ensuring plain text output for systems that don't support Markdown.
